    Don't Panic

    Don't Panic is a great travel companion to TWers on the move. It tells you where all the offices are, how to get there, where you'll be staying, where to eat and hangout. If you ever run into trouble, it can quickly get you in touch with the local admins and legal team too.

    Features:

    • Office listings
    • Admin info
    • Accommodation details
    • Public places
    • Web & mobile (native and browser)
    • Wiki style read + write to AllTW.

    Stack:

    • RoR Web service
    • Angular JS + Twitter Bootstrap web app - both desktop and mobile
    • iOS app
    • Android app
